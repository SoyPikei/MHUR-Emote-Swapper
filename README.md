# Usage
- Open MHUR Emote Swapper.exe
- Choose the mod file which you want to modify
- Choose the character you want the emote to be over
- Choose the slot you want the emote to be over
- Save the new pak file

# Dependencies
- UAssetAPI
- UnrealPak
